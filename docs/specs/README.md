# Specs

このディレクトリには、Unity Mobile Haptics ライブラリの仕様書を配置する。

## 目的

- 実装前に責務、公開 API、プラットフォーム差異、制約事項を明文化する
- 実装中およびレビュー時の判断基準を共有する
- 将来の機能追加時に互換性と設計意図を追跡できるようにする

## 文書一覧

- `unity-mobile-haptics-implementation-spec.md`
  - ライブラリ本体の実装仕様
- `unity-mobile-haptics-class-structure.md`
  - ライブラリ実装時のクラス構成案

## 運用方針

- 仕様変更を伴う実装では、必要に応じて関連する仕様書を先に更新する
- 未確定事項は削除せず、`TODO:` または `TBD:` として残す
- API 名、クラス名、ファイル構成は実装に合わせて継続的に更新する
